# Money Box

Money Box is a web application designed to help users manage their monthly budgets, track savings goals, and handle financial transactions efficiently.

## Live Demo

Check out the live application here: [Money Box](https://ssb2002s.github.io/Money-Box/)

---

## Features

- Add, edit, and track savings goals
- Manage monthly budgets and allocate funds
- Record financial transactions (income/expenses)
- Dashboard with clear financial summaries and insights
- Modern and user-friendly interface

## Getting Started

Follow these steps to run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/ssb2002s/Money-Box.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Money-Box
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
